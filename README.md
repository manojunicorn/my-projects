# Python Calculator

A simple command-line calculator application built using Python.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Handles divide-by-zero case

## Technologies Used
- Python

## How to Run
1. Install Python
2. Run the file:
   python calculator.py
